# MY TODO LIST (Frontend)

> 준비물: Node.js, npm (또는 yarn)

## 시작하기


### 1. 프로젝트 클론  

```
git clone https://github.com/leeejjju/my_todo.git

cd todo-frontend
```

<br>

### 2. 의존성 설치

```
npm install 
```

또는...
```
yarn install 
```

<br>

### 3. 서버 실행 

```
npm start
```
또는...
```
yarm start 
```

<br>

백엔드 서버와 DB, kafka 등이 실행중인지 확인한 후 사용하세요!
